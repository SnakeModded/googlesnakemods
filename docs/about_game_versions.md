# About Game Versions

The web version of the mod loader allows you to choose which version of Google Snake you want to play. While the versions are mostly similar, this feature is especially useful because it lets you use mods that may not yet be updated for the latest version of the game.

## Available Versions

- **Version 1** – Latest dice count  
- **Version 2** – Shield mode added  
- **Version 3** – Arrow mode added (Includes minor fixes for the board size bug, Fixes dice–shield softlock issue) 
- **Version 4** – Daily Challenge added (Larger default game size)  
- **Version 5** – Touchpad controls added for mobile devices  
- **Version 6** – Hotdog mode added  
- **Version 7** – Magnet mode added  
- **Version 8** – Gate mode added  
- **Version 9** – Avocado and mango fruits added  
- **Version 10** – Additional apple count options added
