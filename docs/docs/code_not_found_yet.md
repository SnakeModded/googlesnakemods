# Code Not Found Error

If you encounter the **"Code Not Found"** error after a few seconds while playing the game, follow the steps below to fix it.

## Steps to Resolve

1. Check the game mode you are trying to play.
2. Ensure the same mode is selected and active on your display/settings.
3. If the error still appears:
   - Click on **Change Mode**
   - Select the mode you want to play
4. Confirm the selection and restart the game if required.

## Result

Once the correct mode is selected, the **Code Not Found** error should be fixed.
